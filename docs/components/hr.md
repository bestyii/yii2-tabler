# Hr

`bestyii\tabler\Hr` renders a labeled divider using Tabler's `hr-text` styles.

## Demo

```php-demo
use bestyii\tabler\Hr;

echo Hr::widget([
    'text' => 'Quick toggle',
    'position' => 'center',
    'color' => 'muted',
]);
```
