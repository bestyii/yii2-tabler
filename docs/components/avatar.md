# Avatar

`bestyii\tabler\Avatar` renders a Tabler avatar with image, initials, icon and status dot support.

## Demo

```php-demo
use bestyii\tabler\Avatar;

echo Avatar::widget([
    'text' => 'JD',
    'color' => 'blue',
    'status' => 'success',
]);
```
